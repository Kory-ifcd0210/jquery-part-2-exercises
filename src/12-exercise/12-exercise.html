Instructions

Use the jQuery width() and height() methods to read the values of the content width and height of the .box element.

Set the text content of the .box element to be the width and height of the element. If it has a width of 200 and a height of 100, the text content should be 200x100 "width" + "x" + "height"

Each time the button is clicked you should change the text content to be the current width and height and after that you should increase the width and the height by 20 so that the next time it is clicked the dimensions are larger.

Note that the method returns a number without the unit.

The height() method returns a number that might be a float, so you should use the parseInt() JavaScript method to convert the value to a integer.
