Instructions

Use the jQuery animate() method to animate the height of the .box element when the button is clicked.

The animate() method should use the following options:

  • height: "toggle"
  • duration: 1000ms
  • easing: "swing"

When using a toggle option in the height css property that we want to animate, the function will read the previous height value and then it will collapse the height to 0.

A second execution of the method with the option of toggle will restore the initial height value and set the display back to block.
