Instructions

Use the jQuery height() method to read the value of the content height of the .box element.

Create a new p element with a text content of the height value from the height() method.

Add the p element before the .box element when the Show height button is clicked, only once. If the element exists, it should just change the value of the p element.

Each time the button is clicked you should add 20 to the current height of the .box element.

The text content of the p element should be the value before it was increased.

Note that the method returns a number without the unit.

The height() method returns a number that might be a float, so you should use the parseInt() JavaScript method to convert the value to a integer.
