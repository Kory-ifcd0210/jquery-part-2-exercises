Instructions

Use the jQuery animate() method to animate the height of the .box element when the button is clicked.

The animate() method should use the following options:

  • height: 200px
  • duration: 300ms
  • easing: "linear"

Example use of the jQuery animate() function.


$(".box").animate(
  {
    cssPropertyKey: cssPropertyValue,
  },
  duration,
  "easing"
);
