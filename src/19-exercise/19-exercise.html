Instructions

Use several jQuery animate() methods to animate the .img element when the button is clicked.

The animate() method can be chained to execute several animate() methods one after the other.

animate(options).animate(options).animate(options)...

Add the following animations in the order they appear. (width then height then padding)

  • Animation 1:width: 24em, duration: 1200ms
  • Animation 2:height: 16em, duration: 100ms
  • Animation 3:padding: 2em, duration: 1000ms

You should also add a last parameter to the animate() method that is a callback function which will be executed when each animation ends.

In the function you should create a new p element that has a text content of Finished animating: animation value. The "animation value" would be the animation value you had to animate in each animate() function.

Append each p element as the last child of the .ex-wrapper div element.

Example: 


$("#book")
.animate({ opacity: 0.5 }, 200, function () {
  // Create and add element to the DOM
})
.animate({ padding: "20px" }, 200, function () {
  // Create and add element to the DOM
});

Creates a p element: <p>Finished animating: opacity</p>
